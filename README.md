
一个使用了基于vue2 + mintUI + vuex + asxio + sass 项目

#### 注意：由于涉及大量的 ES6/7 等新属性，node 需要 6.0 以上版本 

//这里使用淘宝镜像安装能快一点
npm --registry https://registry.npm.taobao.org install

### 本地调试模式
```
npm start

```
### 生产模式(部署)

```
npm run build

```
